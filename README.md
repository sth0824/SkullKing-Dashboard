# Skull King 점수판 (모바일 웹)

브라우저에서 **입찰(예측)**·**실제 트릙**을 넣으면 **라운드 점수·총점**을 자동으로 계산합니다. 데이터는 이 브라우저 **로컬 저장**됩니다.

## 요구 사항

- [Node.js](https://nodejs.org/) 20+ 권장

## 실행

```bash
npm install
npm run dev
```

같은 Wi-Fi의 스마트폰에서 열려면(방화벽에 따라 허용이 필요할 수 있음):

```bash
npm run dev:host
```

터미널에 나온 LAN 주소(예: `http://192.168.x.x:5173`)로 접속합니다.

## 빌드

```bash
npm run build
```

`dist` 폴더를 Vercel·Netlify·GitHub Pages 등 **정적 호스팅**에 올리면 됩니다.

## 테스트

```bash
npm test
```

## 점수(구현한 규칙)

- 0트릙 입찰: 실제 0 → +라운드×10, 아니면 −라운드×10
- 1+ 입찰: 맞으면 +입찰×20, 틀리면 −|입찰−실제|×10
- 입찰이 맞을 때만 **인어→스컬킹**, **스컬킹으로 잡은 해적 수** 보너스 적용
- **수동 점수** 켜면 그 라운드는 계산 대신 **직접 점수**로 덮어씀
