# Skull King 점수판 (모바일 웹)

브라우저에서 **입찰(예측)**·**실제 트릭**을 넣으면 **라운드 점수·총점**을 자동으로 계산합니다. 데이터는 이 브라우저 **로컬 저장**됩니다.

처음 쓰실 때는 앱 상단 **📖 이용 안내** 버튼으로 사용 순서·헤더 아이콘 설명을 확인할 수 있습니다.

## 요구 사항

- [Node.js](https://nodejs.org/) 20+ 권장

## 실행

```bash
npm install
npm run dev
```

같은 Wi-Fi의 스마트폰에서 열려면(방화벽에 따라 허용이 필요할 수 있음):

```bash
npm run dev:host
```

터미널에 나온 LAN 주소(예: `http://192.168.x.x:5173`)로 접속합니다.

## 빌드

```bash
npm run build
```

`dist` 폴더를 Vercel·Netlify·GitHub Pages 등 **정적 호스팅**에 올리면 됩니다.

## 테스트

```bash
npm test
```

## 점수(구현한 규칙)

- 0트릭 입찰: 채점용 실제 0 → +라운드×10, 아니면 −라운드×10
- 1+ 입찰: 맞으면 +입찰×20, 틀리면 −|입찰−채점용 실제|×10
- **채점 보정**(해적 등): 물리 트릭 합 검증은 `실제` 그대로, 점수 계산만 `실제+보정` 사용
- 1+ 입찰 성공 시에만: 임의 **추가 보너스** 점수, **1등 항해사**(+30), **데이빗 존스 해양생물**(장당 +20), 확장 보너스
  - **인어→스컬킹**(+40), **해적→인어**(장당 +20), **스컬킹→해적**(장당 +30)
  - **일반 14**(장당 +10), **검정 14**(장당 +20), **Loot 동맹**(회당 +20)
  - **Rascal wager**(10/20): 입찰 성공 시 +, 실패 시 −
- **크라켄** 라운드 체크 시: 전원 `실제` 트릭 합이 라운드와 다를 수 있음(경고 생략)
- **수동 점수** 켜면 그 라운드는 계산 대신 **직접 점수**로 덮어씀
